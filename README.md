Tech Stack: Node.js, Express.js, MongoDB, React.js, JavaScript (ES6+), HTML5, CSS3

Full-Stack Development: Engineered a complete client-server web application to provide a simple and user-friendly platform for browsing and managing books. The project demonstrates integration between frontend and backend systems.

RESTful Backend Server: Built a backend using Node.js and Express.js, creating RESTful API endpoints for handling user authentication and book data. Implemented secure password storage using bcrypt and ensured smooth communication between client and server.

Dynamic Frontend Interface: Developed a responsive frontend using React.js. Implemented reusable components and used Axios to make API calls for fetching and displaying book data dynamically without page reloads.

Database Management: Designed and integrated a MongoDB database to store user and book information. Managed data efficiently for operations like adding, retrieving, and displaying books.
